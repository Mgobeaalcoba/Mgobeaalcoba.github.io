f:T1315,<h2>La Productividad con GenAI</h2>
<p>ChatGPT no es un chatbot: <strong>es tu asistente técnico senior</strong>.</p>
<p>Estos 15 prompts me ahorran <strong>10+ horas semanales</strong>.</p>
<h2>1. Code Review Automático</h2>
<pre><code>Revisá este código Python y encontrá:
1. Bugs potenciales
2. Security issues
3. Performance problems
4. Mejoras de legibilidad

[pega tu código aquí]

Dame feedback en bullets puntuales.
</code></pre>
<h2>2. Escribir Tests</h2>
<pre><code>Generame pytest tests para esta función:

[pega función]

Incluí:
- Happy path
- Edge cases
- Error handling
- Mocking de llamadas externas
</code></pre>
<h2>3. Documentación de Código</h2>
<pre><code>Generame docstring completo para esta función en formato Google:

[pega función]

Incluí:
- Args con tipos
- Returns
- Raises
- Ejemplo de uso
</code></pre>
<h2>4. Optimizar SQL</h2>
<pre><code>Optimizá esta query de BigQuery:

[pega query]

Explicame:
1. Qué está mal
2. Query optimizada
3. Por qué es mejor
4. Estimación de ahorro de costos
</code></pre>
<h2>5. Debugging Assistant</h2>
<pre><code>Tengo este error:

[pega stack trace]

Mi código:
[pega código relevante]

Ayudame a:
1. Identificar la causa root
2. Proponer solución
3. Prevenir que pase de nuevo
</code></pre>
<h2>6. Traducir Requisitos a User Stories</h2>
<pre><code>Convertí este requisito vago a user stories INVEST:

Requisito: "Necesitamos un dashboard de ventas"

Dame:
- User stories específicas
- Acceptance criteria
- Estimación de complejidad
</code></pre>
<h2>7. Generar Data Sintética</h2>
<pre><code>Generame un dataset CSV de 100 filas con estas columnas:

- transaction_id (UUID)
- user_id (integer 1000-9999)
- amount (float 100-10000)
- currency (ARS, USD, BRL con distribución 70%, 20%, 10%)
- date (entre 2026-01-01 y 2026-05-31)
- status (completed, pending, failed con distribución 85%, 10%, 5%)

Datos realistas, no repetitivos.
</code></pre>
<h2>8. Regex Helper</h2>
<pre><code>Necesito un regex para validar:

- Email válido
- Teléfono argentino formato +54 9 11 1234-5678
- DNI argentino (7-8 dígitos)

Dame:
1. El regex
2. Ejemplos que matchean
3. Ejemplos que NO matchean
4. Código Python para usar
</code></pre>
<h2>9. Refactoring Suggestions</h2>
<pre><code>Este código funciona pero es feo:

[pega código]

Refactorizalo siguiendo:
- Clean Code principles
- SOLID principles
- Python best practices

Explicame los cambios.
</code></pre>
<h2>10. Error Messages Amigables</h2>
<pre><code>Convertí este error técnico en mensaje user-friendly:

Error: "psycopg2.OperationalError: could not connect to server: Connection refused"

Dame:
1. Mensaje para usuario final
2. Mensaje para logs técnicos
3. Acciones sugeridas
</code></pre>
<h2>11. Naming Things</h2>
<pre><code>Tengo que nombrar:

- Función que calcula impuesto a las ganancias mensual
- Variable que guarda el monto neto después de descuentos
- Clase que maneja simulación de salario

Dame 3 opciones para cada una en inglés, siguiendo:
- Python conventions
- Clean Code
- Descriptive but concise
</code></pre>
<h2>12. Meeting Notes → Action Items</h2>
<pre><code>Convertí estas notas caóticas de reunión en action items:

[pega notas]

Dame:
- Action items claros con owner
- Deadlines sugeridos
- Priorización (P0, P1, P2)
- Formato Markdown
</code></pre>
<h2>13. Generar Fixtures de Test</h2>
<pre><code>Generame fixtures de pytest para testear esta función:

[pega función]

Incluí casos:
- Usuario argentino con cónyuge
- Usuario sin deducciones
- Usuario con sueldo en mínimo no imponible
- Usuario con sueldo alto (top bracket)
</code></pre>
<h2>14. Configuración de Herramientas</h2>
<pre><code>Generame:

1. .gitignore para proyecto Python con:
   - Jupyter notebooks
   - Virtual environments
   - Data files (CSV, Parquet)
   - Secrets (.env)
   - IDE configs

2. requirements.txt con versiones específicas para:
   - pandas, numpy, requests
   - testing (pytest, pytest-cov)
   - linting (black, flake8)
</code></pre>
<h2>15. Onboarding Docs</h2>
<pre><code>Generame README.md para este proyecto:

[descripción breve del proyecto]

Incluí:
- Descripción clara
- Prerequisites
- Setup instructions paso a paso
- Cómo correr tests
- Cómo deployar
- Troubleshooting común
- Contacto para preguntas
</code></pre>
<h2>Meta-Prompt: Mejorar tus Prompts</h2>
<pre><code>Analizá este prompt y mejoralo:

[tu prompt original]

Dame:
1. Version mejorada del prompt
2. Qué le faltaba al original
3. Por qué la nueva versión es mejor
</code></pre>

<p>GenAI no reemplaza a developers: <strong>los amplifica</strong>.</p>
<p>Estos 15 prompts son mi "toolkit" diario.</p>
<p>Adaptalos a tu workflow y multiplicá tu productividad.</p>

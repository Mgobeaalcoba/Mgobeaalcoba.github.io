<p>La mayoría usa ChatGPT API así:</p>

<h2>Truco #1: Function Calling</h2>
<p>Hacer que GPT devuelva <strong>structured data</strong>.</p>
<pre><code class="language-python">functions = [{
    "name": "extract_transaction",
    "description": "Extrae datos de una transacción del texto",
    "parameters": {
        "type": "object",
        "properties": {
            "amount": {"type": "number"},
            "currency": {"type": "string"},
            "date": {"type": "string", "format": "date"}
        },
        "required": ["amount", "currency"]
    }
}]

response = openai.ChatCompletion.create(
    model="gpt-4",
    messages=[{"role": "user", "content": "Pagué $5000 USD el 3 de mayo"}],
    functions=functions,
    function_call="auto"
)

# Response es JSON estructurado
{
    "amount": 5000,
    "currency": "USD",
    "date": "2026-05-03"
}
</code></pre>
<h2>Truco #2: Seed para Reproducibilidad</h2>
<pre><code class="language-python">response = openai.ChatCompletion.create(
    model="gpt-4",
    messages=[...],
    seed=42,  # Mismo seed = misma respuesta
    temperature=0
)

# Útil para testing y debugging
</code></pre>
<h2>Truco #3: JSON Mode (Forzar JSON)</h2>
<pre><code class="language-python">response = openai.ChatCompletion.create(

        "role": "user",
        "content": "Extrae nombre y email de: Juan Pérez - juan@email.com"
    }],
    response_format={"type": "json_object"}
)

# Garantizado que devuelve JSON válido
</code></pre>
<h2>Truco #4: Streaming para UX</h2>
<pre><code class="language-python">stream = openai.ChatCompletion.create(

    stream=True
)

for chunk in stream:
    if chunk.choices[0].delta.content:
        print(chunk.choices[0].delta.content, end="")

# Usuario ve respuesta aparecer en tiempo real
</code></pre>
<h2>Truco #5: Token Counting Exacto</h2>
<pre><code class="language-python">import tiktoken

encoding = tiktoken.encoding_for_model("gpt-4")

def count_tokens(text):
    return len(encoding.encode(text))

prompt = "Tu prompt largo aquí..."
tokens = count_tokens(prompt)
cost_estimate = tokens / 1000 * 0.03  # GPT-4 input cost

print(f"Tokens: {tokens}, Costo estimado: ${cost_estimate:.4f}")
</code></pre>

<h2>Truco #7: Prompt Caching (Automático en GPT-4 Turbo)</h2>
<pre><code class="language-python"># Prefijo del prompt (cached automáticamente)
system_prompt = """Sos un experto en finanzas argentinas.
Tus respuestas deben ser precisas y citar fuentes.
[... 5000 tokens de contexto ...]"""

# Esto se cachea. Requests subsecuentes son 50% más baratos.
response = openai.ChatCompletion.create(
    model="gpt-4-turbo",
    messages=[
        {"role": "system", "content": system_prompt},
        {"role": "user", "content": "Pregunta corta"}
    ]
)
</code></pre>
<h2>Truco #8: Logprobs para Confidence</h2>
<pre><code class="language-python">response = openai.ChatCompletion.create(

# Ver probabilidad de la respuesta

</code></pre>
<h2>Truco #9: Vision API para OCR</h2>
<pre><code class="language-python">def extract_text_from_image(image_url):
    response = openai.ChatCompletion.create(
        model="gpt-4-vision",
        messages=[{
            "role": "user",
            "content": [
                {"type": "text", "text": "Extrae el texto de esta imagen en JSON"},
                {"type": "image_url", "image_url": image_url}
            ]
        }],
        max_tokens=1000
    )
    return response.choices[0].message.content
</code></pre>
